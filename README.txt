זהו קובץ README שמתאר מהו הפרויקט ואיך להפעיל אותו.

--לכל הפקודות להפעלת הפרויקט נא לרדת לתחתית העמוד.

מטרת הפרויקט:
אתר שמטרתו היא להוסיף כמויות עצומות של אנשי קשר לכל טלפון שאנשי הקשר שלו מסונכרנים עם חשבון הגוגל שלו בעזרת הAPI של google People.
כמו כן חשבון הגוגל שאליו מסונכרנים אנשי הקשר (שאמורים להופיע באתר google contacts) חייב לפתוח פרויקט כאן
https://console.cloud.google.com/apis/dashboard
בדרך הבאה:
1. צרו פרויקט חדש.
2. לכו ל-Library, חפשו google people api ותורידו אותו לפרויקט (האופציה הראשונה בחיפוש).
3. לכו לcredentials לחצו על create credentials, ותיצרו OAuth Id.
4. כשאתם מתבקשים לכתוב את סוג האפליקציה לחצו על Desktop App ובחרו שם.
5. תורידו עכשיו את הקובץ Client JSON.
6. שנו את השם שלו ל"client_secret" עם סיומת json.
7. מקמו את הקובץ בתיקייה website/resources.

לאחר מכן עקבו אחר ההוראות פה.

הקבצים שקיימים בפרויקט:
"main.py" - הקובץ פייתון הראשי, ממנו הכול מתחיל, אם נריץ אותו הוא זה שיפעיל את הפרויקט.
"README.txt" - קובץ הסבר על הפרויקט.
"requirments.txt" - הקובץ שמכיל את כל הספריות הדרושות לאינטרפרטר של פייתון כדי שהפרויקט יעבוד.
"Tests - xlsx" - תיקיית טסטים שמכילה קבצי אקסל לדוגמה.
"website" - תיקייה שמכילה את קבצי האתר.
	תחת website:
	"init.py" - קובץ אתחול של הספרייה בפייתון בשם flask.
	"navigation.py" - קובץ פייתון שאחראי על הניתוב באתר.
	"google_contacts.py" - קובץ הפייתון החשוב ביותר, הוא המוח של הפרויקט והוא אחראי על כל ההוספה של אנשי הקשר והתעסקות עם הAPI.
	"resources" - תיקייה שמכילה את קובץ הclient_secret.json בעצם הקובץ שאחראי על המשתמש גוגל שאליו יכניסו אנשי קשר.
	"static" - התיקייה שמכילה את את הפונטים, התמונות וה-css של כל האתר.
	"templates" - תיקייה שמכילה את דפי ה-html של הפרויקט.
	"uploads" - תיקייה ריקה שאמורה להכיל את קובץ האקסל שמישהו בוחר ובזמן שהסקריפט רץ אז הוא קורא את הקובץ אקסל בתיקייה, לוקח את המידע ואז מוחק את הקובץ אקסל מהתיקייה, לכן הוא תמיד יראה ריק.
	
שלבים ופקודות להפעלת הפרויקט:
1. כניסה לתיקיית הפרויקט "create-contacts"
2. python -m venv venv -> יצירת סביבה וירטואלית.
3. אם מערכת ההפעלה היא לינוקס תריצו את הפקודה:
	source venv/Scripts/activate
אם מערכת ההפעלה היא  windows תריצו את הפקודה:
	venv\Scripts\activate

4. pip install -r requirments.txt
5. כאן צריך להריץ את הקובץ main.py, אם תריצו את הפקודה בתיקייה הראשית:
	python main.py
זה יריץ את האתר על הכתובת 0.0.0.0:5000, צריך לפתוח את הקובץ פייתון main.py כדי לשנות את הhost ואת ה-port.
